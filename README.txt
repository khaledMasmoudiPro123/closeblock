Close block - is a module which allows to add a "close" button to any block.
Features:
- You can choose an effect (None, SlideUp and FadeOut)
- Effect duration

Installation:
1. Install the module
2. Configure it on the theme page
3. Activate it on the Blocks settings page
4. Check the permissions page
